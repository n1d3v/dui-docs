---
title: Content
---

The `content` attribute is the content / text of an element. 

### Examples
It is primarily used to show text on an element, some examples are:

```xml
<CCPushButton content="Push button" />
```
> Push button example, can be used on other buttons.

```xml
<element content="Hello world!">
```
> Element / Text label example, bare minimum of text.
