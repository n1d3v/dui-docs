---
title: <duixml>
---

`<duixml>` is the start and end of any DirectUI code written. It is the root element that defines items inside of the DirectUI code. It is like `<html>` in HTML.

### Examples
You can use `<duixml>` as the root element like:

```xml
<duixml>
    <!-- Elements -->
</duixml>
```

### Parameter
There are no parameters available for `<duixml>`.

### Notes
There are no notes available for `<duixml>`.
